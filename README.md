- Градов В.М.

## Цель курса

Цель курса заключается в том, чтобы научиться формулировать и решать дифференциальные уравнения с помощью численных методов. Курс направлен на освоение различных алгоритмов - метода Галёркина, схем Эйлера, Рунге–Кутты, метода прогонки и других, для задач с разными типами краевых условий, включая многомерные модели. Работа в рамках курса развивает понимание того, как математические модели переводятся в вычислимую форму, как выбирать подходящий численный метод и как анализировать точность и устойчивость полученного решения.
